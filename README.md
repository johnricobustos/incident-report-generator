# Northwind Incident Report Generator

An AI-powered web tool for HR personnel to draft professional incident report emails (Notices to Explain) based on employee violations.

> 🚧 **Work in progress** — full app coming soon.

## Tech Stack
- HTML / CSS / JavaScript
- Tailwind CSS
- Claude API (Anthropic)
- GitHub Pages (hosting)
